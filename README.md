# Vault Configuration

This repository allows to update vault configuration and vault policies.

Start projects in python virtual environment:

~~~
source bootstrap.sh
~~~

Go out:

~~~
deactivate
~~~

## Test

Run `scripts/tests.sh` or `scripts/tests.sh --interactive`.
